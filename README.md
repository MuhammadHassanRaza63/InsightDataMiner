**InsightDataMiner** is an intelligent, 100% No-Code Data Science workspace. It empowers users to seamlessly transition from chaotic, raw datasets to crystal-clear visual analytics and deployed Machine Learning models—without writing a single line of code. 

Designed for beginners but powered by advanced AI, IDM automates hours of data processing into a few simple clicks.

## ✨ Core Features & Platform Modules

### 🛠️ Module 1: Smart Data Preprocessing
Upload your raw data (`.csv`, `.xlsx`) and let the intelligent engine do the heavy lifting.
* **Auto-Imputation:** Automatically handles missing values (Median for numeric, Mode for categorical).
* **Outlier Mitigation:** Detects and clips extreme anomalies using the Interquartile Range (IQR) method.
* **Feature Encoding & Scaling:** Applies Label Encoding for text data and Z-Score standardization for numeric features.
* **Refinement Report:** Provides a comprehensive "Before & After" health check, complete with distribution visualizations and clean data export.

### 📊 Module 2: Visual Analytics Dashboard
Turn boring numbers into stunning interactive graphics.
* **Auto-Executive Dashboard:** One-click heuristic dashboard generation summarizing key KPIs and feature distributions.
* **Manual Chart Builder:** Create Scatter plots, Bar charts, Histograms, Box plots, Pie charts, and Correlation Heatmaps using Plotly.
* **HD Export:** Pin your favorite charts to a custom board and export the entire dashboard as a high-resolution image.

### 🤖 Module 3: AutoML Predictions
Evaluate your dataset and train predictive models automatically.
* **Algorithm Race:** Automatically trains and compares multiple models (Logistic/Linear Regression, Decision Trees, Random Forests) based on your selected target and task type (Classification or Regression).
* **Explainable AI:** Highlights Feature Importance to show which variables impact the AI's decisions the most.
* **Live Sandbox:** Test the winning model instantly using an AJAX-powered real-time prediction form.
* **Model Deployment:** Export the fully trained model and its encoders as a `.pkl` file for external deployment.

### 🎨 Enterprise Theme Engine
Includes a custom-built, cookie-based dynamic theme switcher that seamlessly syncs the entire UI, including Python-generated Matplotlib and Plotly charts, across three modes:
* **Original Dark:** Deep-sea, futuristic neon look.
* **Enterprise Light:** Sleek, minimal, print-friendly aesthetic.
* **Modern Hybrid:** Premium frosted-glass corporate design.

## 💻 Tech Stack

* **Backend:** Python, Django
* **Data Processing & ML:** Pandas, NumPy, Scikit-Learn
* **Data Visualization:** Matplotlib, Seaborn, Plotly
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5
* **Interactivity:** AJAX, SweetAlert2, html2canvas, Particles.js
